---III.Требования к запуску и установка---

Работающая игра Gnomoria версии 0.8.24 (возможны другие версии, но их работа не гарантирована)

Системные требования игры и данной программы совпадают.

Требуется .NET Framework следующих версий:
.NET Framework 4.0 Client Profile:		http://www.microsoft.com/en-us/download/details.aspx?id=24872
.NET Framework 3.5 SP1 (Если у вас Windows XP): http://www.microsoft.com/en-us/download/details.aspx?id=22



---IV. Установка---

Установите утилиту в папку с установленной игрой, где находится Gnomoria.exe




---V. Использование---

1. Перед запуском программы, выйдите из игры.

2. Запустите GnomeExtractor.exe

3. Откройте файл сохранения с помощью данной программы.

4. Внесите необходимые вам изменения.

5. Сохранитесь. (При сохранении, изменения записываются в исходный файл)
